这个模板的出发点不是“再做一个首页”，而是把你的个人站点做成一个持续生长的工作台。

你可以在这里同时展示：

- 长文博客
- Obsidian 同步过来的研究笔记
- GitHub 项目和近期动态
- 现在正在做什么

## 为什么是这种界面

`4router.net` 的前端有一种很鲜明的质感：像素字、浅暖底色、玻璃卡片、信息密度高但并不压抑。  
我把这套语言转成了更适合个人博客的版本，让阅读和展示都更舒服。

## 后续可以继续扩展

- 接入评论系统
- 生成文章封面图
- 自动从 GitHub Actions 同步 Obsidian 导出目录
- 增加多语言页面
